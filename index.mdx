# Commerce BD Documentation

BigBoss Commerce is a multi-branch ERP designed for Bangladesh retail — single tenant, many stores, full warehouse management, NBR-compliant accounting.

These guides walk you through setup and daily operations for each module. Written for store owners, managers, accountants, cashiers, and warehouse staff — no engineering background required.

## Modules

### Setup
- **[Platform Settings](/commerce-bd/docs/platform/getting-started)** — store config, branches, users, payment methods, VAT
- **[Products](/commerce-bd/docs/products/getting-started)** — catalog, variants, categories, size guides, reviews

### Operations
- **[Customers](/commerce-bd/docs/customers/getting-started)** — profiles, addresses, credit terms, A/R ledger
- **[Suppliers](/commerce-bd/docs/suppliers/getting-started)** — vendors, tax compliance, A/P ledger
- **[Inventory](/commerce-bd/docs/inventory/getting-started)** — stock, transfers, purchases, lots, RMA, warehouse tasks
- **[Warehouse (Advanced)](/commerce-bd/docs/warehouse-advanced/getting-started)** — cycle counting, lots, procurement, replenishment, cost layers, dispatch, quality
- **[Sales](/commerce-bd/docs/sales/quotations-overview)** — quotations, orders, fulfillment
- **[POS](/commerce-bd/docs/pos/getting-started)** — point-of-sale terminal, shifts, cash reconciliation

### Finance & Growth
- **[Accounting](/commerce-bd/docs/accounting/getting-started)** — chart of accounts, A/P, A/R, Mushak filings, day close, reports
- **[Promotions](/commerce-bd/docs/promotions/getting-started)** — discount programs, vouchers, cart evaluation
- **[Loyalty](/commerce-bd/docs/loyalty/getting-started)** — earning rules, tiers, redemption, referrals
- **[CRM](/commerce-bd/docs/crm/getting-started)** — leads, opportunities, accounts, pipelines (SDK ready, UI in development)

## How to use these docs

Every guide links to the matching dashboard page. Open the dashboard in another tab and follow along — you'll be set up in minutes.

If something behaves differently from what's described, [open an issue on GitHub](https://github.com/classytic/commerce-bd-docs/issues).
