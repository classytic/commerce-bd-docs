# Getting Started

First-time setup when you launch your store. Takes about 10 minutes.

---

## 1. Set Your Store Name and Currency

Go to **Settings > Platform Config** and fill in:

- **Platform Name** — appears on invoices, emails, and the dashboard header
- **Currency** — Bangladesh Taka is the default; change it if you operate elsewhere

Click **Save**. These values are visible to every staff member and on every customer-facing document.

[Open Platform Config](/dashboard/platform-config)

---

## 2. Configure Your Fiscal Year

Bangladesh's standard fiscal year runs **July to June**. Most businesses keep this default.

Go to **Accounting > Fiscal Periods** and confirm the start month. If you operate on a calendar year (January start), change it now — once you start posting invoices, switching the fiscal year requires a migration.

[Manage Fiscal Periods](/dashboard/accounting/fiscal-periods)

---

## 3. Add Your Branches

Even if you only have one store, you need at least one branch. Go to **Settings > Branches** and add each physical location (store, warehouse, outlet).

Branches isolate stock, staff, and POS shifts. The first branch you create becomes the default.

[Manage Branches](/dashboard/branches)

---

## 4. Create Staff Accounts

Go to **Settings > Users** and add an account for each staff member. Each user gets:

- An email + password (used to log in)
- A **system role** like `cashier` or `store-staff`
- Membership in one or more branches

Only **superadmin** users can create accounts — make sure the first owner-account has that role.

[Manage Users](/dashboard/users)

---

## 5. Configure Payment Methods

Tell the system which payment methods you accept (cash, bKash, bank transfer, card). The POS and checkout will only show options you've enabled.

[Configure Payment Methods](/dashboard/platform-config) — see [Payment Methods guide](/dashboard/platform/learn/payment-methods)
